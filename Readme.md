# Учебный проект "Cinemaddict"

## Стек: 
1. JS (ES6)
2. webpack
3. [dayjs](https://day.js.org/)
4. [chartjs](https://www.chartjs.org/)

## Паттерны: 
1. Паттерн проектирования <b>MVP</b>
2. Поведенческий паттерн <b>Observer</b>
3. Структурный паттерн <b>Adapter</b>
